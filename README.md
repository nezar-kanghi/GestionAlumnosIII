En este ejrcicio gestinamos notas de un alumno a partir de un fichero alumnos.txt, donde cada linea tiene el formato nombre;nota. Primero lee el archivo con BufferedReader, separa los datos con split(";") y utilizamos un HashMap<String, List<Double>>para agrupar todas las notas de cada alumno. Convierte las notas a double con Double.parseDouble() para poder calcular la media, luego recorre el mapa para calcular la nota media de cada alumno, guarda en otro hashmap unicamente a los aprobados y al mismo tiempo determina cual es el alumno con mejor media. Finalmente escribe en el fichero resultado.txt el listado de alumnos aprobados con su media y el nombre del alumno con mejor nota, utilizando BufferedWriter
